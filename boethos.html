Boēthos™

Race Engineer in a Box

Boēthos is the software side of the ERX ecosystem — built to help drivers review runs, understand data, set course gates, and improve faster.

View Status

The Purpose

Know more. Drive better.

Boēthos turns raw data into useful feedback. The goal is not just to record a run, but to help a driver understand where time was gained, where time was lost, and what to focus on next.

Built to work directly with ERX hardware — load a run, review the data, compare to your previous best.

Software Preview

What You'll See

Boēthos transforms raw ERX logger data into clear visual feedback — every run, every sector, every decision point.

Overview Dashboard

Event summaries, session statistics, and key performance metrics in one place.

Data Analysis

GPS paths, speed traces, G-forces, and logged events from every run.

Precision View

Overlay multiple runs, inspect racing lines, and visualize where time was gained or lost.
