Black Box

Grassroots Driver Data System

A vehicle-mounted motorsport data logger built for GPS logging, IMU data, Bluetooth control, SD card recording, and Boēthos software integration.

View Development Status

Driver Data Without the Complication

Black Box was created to give grassroots drivers useful data without needing a professional race engineering setup. The goal is simple: record the run, understand the car, and make better decisions from real information.

Why Black Box Exists

Most grassroots drivers make setup changes based on feel alone. Feel matters, but data helps confirm what the car actually did. Black Box is being developed around real autocross testing, real hardware limitations, and real driver needs.

It is not meant to replace the driver. It is meant to help the driver see the run more clearly after the helmet comes off.

Core Features

GPS Logging

Position, speed, course mapping, start/finish gates, and sector points.

IMU Data

Acceleration, braking, rotation, and movement data for run analysis.

Bluetooth Control

Phone app connection for setup, course points, and future run controls.

SD Card Logging

Run data is recorded locally for review, storage, and future analysis.

Boēthos Ready

Built to work with the Boēthos app and driver review software ecosystem.

Expandable

Future expansion support for ECU data, CAN bridge input, sensors, and accessories.

Current Hardware

Main Controller

Arduino Mega 2560 development platform used for Rev 1 system testing and firmware development.

GPS Module

NEO-M8N GPS receiver operating at 5–10 Hz for position, speed, course mapping, and gate detection.

Motion Sensor

MPU-6050 6-axis IMU providing acceleration and rotational movement data.

Real Time Clock

DS3231 RTC module for accurate timestamps and log file organization.

Data Storage

Micro SD card logging with removable storage for post-run review and analysis.

Connectivity

Bluetooth communication for phone app setup, course configuration, and future run management tools.

System Specifications

GPS Logging

Position, speed, heading, start gate, finish gate, and sector tracking.

IMU Logging

Lateral acceleration, longitudinal acceleration, and rotational movement recording.

File Format

CSV-based logging compatible with Boēthos analysis software and spreadsheet tools.

Course Setup

Bluetooth transfer of start, finish, and sector locations from the companion phone app.

Status Indicators

Dedicated LEDs for logging state, GPS lock, Bluetooth connection, and system status.

Expansion Port

DB37 expansion interface planned for ECU data, CAN bridge support, and future sensors.
