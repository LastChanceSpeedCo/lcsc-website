Black Box

Grassroots Driver Data System

A grassroots motorsport data system built for GPS logging, IMU data, Bluetooth control, SD card recording, and Boēthos integration.

View Status

Driver Data Without the Complication

Black Box was created to give grassroots drivers access to meaningful data without the cost and complexity of professional motorsport systems.

Core Features

GPS Logging

Position, speed, course mapping, start/finish gates, and sector points.

IMU Data

Acceleration, braking, rotation, and movement data for run analysis.

Bluetooth Control

Phone app connection for setup, course points, and future run controls.

SD Card Logging

Run data is recorded locally for review, storage, and future analysis.

Boēthos Ready

Built to work with the Boēthos app and driver review software ecosystem.

Expandable

Future expansion support for nodes, ECU data, sensors, and accessories.

Specifications

GPS

5–10 Hz position and speed logging.

IMU

Motion data for acceleration, braking, and rotation review.

Storage

Micro SD card data logging for run review and analysis.

Connectivity

Bluetooth connection for phone app setup and control.

Power

Designed for vehicle-mounted motorsport use.

Platform

Current development hardware: Black Box Rev 1.
