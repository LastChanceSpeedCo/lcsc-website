What Happens Next

The process

01

Application Review

Vehicle, motorsport experience, and planned use are reviewed to find the right fit for the initial beta group.

02

Personal Follow-Up

If selected, you'll be contacted directly by email to discuss your setup, answer questions, and confirm ERX-L is a good match.

03

Purchase Code

Approved testers receive a private purchase code that unlocks the ERX-L unit in the Last Chance Speed Co. online store.

About the Program

Why it's limited

The current early access program is intentionally small. This allows firmware, hardware revisions, and Boēthos software updates to be validated through real competition while maintaining direct communication with every tester.
